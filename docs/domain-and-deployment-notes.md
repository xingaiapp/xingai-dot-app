# XingAI Dot App Domain And Deployment Notes

Last updated: 2026-05-14

## Canonical URL For This Vercel Project

Use this URL as the stable public preview / project URL:

```text
https://xingai-dot-app.vercel.app/
```

Do not use one-off Vercel deployment URLs as the product URL in docs, product cards, or cross-links.

Examples of one-off deployment URLs:

```text
https://xingai-dot-kkh7xinhk-xingaiapps-projects.vercel.app
```

These URLs are generated by Vercel for individual deployments. They can still exist internally as deployment sources, but they should not be used as the official site URL.

## Alias Policy

The current policy is:

- Use `https://xingai-dot-app.vercel.app/` for the XingAI website.
- Use `https://wear.xingai.app/` for the Wear AI V1 demo.
- Do not point users to generated deployment URLs.

## Commands Used

Assign the stable alias to the latest deployment:

```bash
vercel alias set https://<latest-deployment-url> xingai-dot-app.vercel.app
```

Remove old aliases:

```bash
vercel alias rm xingai.app --yes
vercel alias rm www.xingai.app --yes
vercel alias rm xingai-dot-app-xingaiapps-projects.vercel.app --yes
vercel alias rm xingai-dot-app-xingaiapp-xingaiapps-projects.vercel.app --yes
```

Verify:

```bash
curl -I https://xingai-dot-app.vercel.app/
vercel alias ls
```
