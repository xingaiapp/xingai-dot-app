# Vercel — xingai.app

## Root Directory

Leave **empty** or **`.`** — `package.json` with `next` is at the **repo root**.

If Root Directory is set to a subfolder by mistake, you will see **“No Next.js version detected”**.

## Deploy

- Repo: `xingai-dot-app`
- Production domain: `xingai.app` / `xingai-dot-app.vercel.app`
